Ini adalah demo

ini adalah contoh simpel dari cara untuk pembuatan menu yang menggunakan css dan html tidak memerluankan javasript kara saya yakin jika menggunakan javascript akan memperberat loading halaman anda makan dari itu saya hadirkan yang lebih ringan dan simpel

Menu ini di buat untuk para blogger yang menggunakan versi amp yang di buat sengaja tanpa melibatkan javascript hanya mengandalkan css dan html kalian bisa mencotohkan cara pembuatanya karna dari disain sengaja di buat agar kalian sedikit memahami componen css yang mudah untuk di mengerti

Menu ini sedikit mengadalkan versi canvas yang akan menghasilkan efek yang menawan sehingga ini tidak kalah dengan efek yang di hasilkan oleh javascript dan boostrap karna untuk tahun 2018 ini sudah saatnya kita memiliki cara yang tepat untuk mengandalkan AMP untuk sebuah halaman yang rata rata pengunjung menggunakan versi mobile agar mereka tetap merasa senang
